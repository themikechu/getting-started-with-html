how the web works

A person uses a computer which communicates with servers using HTTP. From their communication, the server then sends information back to the computer. The web browseris how this is viewed.

We learned how to insert links like this, how to make text bold, how to italicize and how to insert an image

We also learned about inline and block elements. Block elements have a "box" around them.


We also learned that if you have an error in your code, you computer does not fix it on it's own. The computer will follow the directions that you give it.
